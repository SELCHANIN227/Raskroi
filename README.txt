ДЕМО-СБОРКА «Раскрой» — работает без интернета и без Supabase.

Как запустить:
1. Распакуй папку.
2. Открой index.html в браузере. На телефоне/компе — всё равно.
   (Если фон/шрифты выглядят чуть иначе — это потому что Google Fonts
    не грузятся офлайн, подставляется системный шрифт. На реальном
    сайте с интернетом шрифт будет родной.)

Что это:
- Полностью рабочий интерфейс на тестовых данных (2 изделия, детали,
  клиент, эскиз). Можно кликать, добавлять, считать раскрой — всё
  хранится в памяти вкладки и сбрасывается при перезагрузке.
- Это ТА ЖЕ сборка, что и боевая, отличие ровно в одном флаге
  (демо-режим включён: fD=true). Никакой логики не менял.

Чем отличается от боевого app.js:
- fD=!0 вместо fD=!1 (включён встроенный мок Supabase из 01_demo-data).
- из index.html убрана регистрация service worker (чтобы локально
  не кэшировалось и не мешало).

НЕ заливай эту папку в прод — это только для просмотра.
